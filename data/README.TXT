ISLEX Recordings (v. 1.0)

This package contains the soundfiles in MP3 format - 64 Kbps, 44.1 KHz, 1 channel, MPEG Audio (Version 1) (Layer 3).

ABOUT.TXT contains further information on the ISLEX project and the ISLEX recordings.

ISLEX.csv contains information on the containts of each sound file. The first field is the name of the file, the second field the Icelandic word read in the file and the third field has grammatical information on the word.
